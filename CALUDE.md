# Resume Site Project

## Goal
Build a resume website hosted on GitHub Pages.
Visitors can view the resume online and download a formatted PDF.

## Stack
- Nextjs
- html2pdf.js for PDF generation (CDN, no build step)
- GitHub Pages for hosting

## Content source
Fetch resume content from https://www.linkedin.com/in/mtablado/, including professional experience, education, skills, awards and recognitions 


## File structure
Organize the project using instructions from Nextjs site documentation at https://nextjs.org/docs/app/getting-started/project-structure adapated to the nature of this project which has no backend (it is purely executed at browser)

## Preferences
- Clean, professional design
- Mobile-friendly
- Print-friendly PDF output (A4)
